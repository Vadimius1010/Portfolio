# Проверка гипотез по увеличению выручки в интернет-магазине - оценка результатов A/B теста

<b> Постановка задачи </b>

Вместе с отделом маркетинга был подготовлен список гипотез для увеличения выручки. Необходимо, используя данные интернет-магазина, приоритезировать гипотезы, произвести оценку результатов A/B-тестирования различными методами.

---

<b>Вывод</b>

В результате работы была проведена приоритизация гипотез по фреймворкам ICE и RICE. Далее произведен анализ результатов A/B-теста, построены графики кумулятивной выручки, среднего чека, конверсии по группам, а посчитана статистическую значимость различий конверсий и средних чеков по сырым и очищенным данным. На основании анализа было принято решение о нецелесообразности дальнейшего проведения теста и о победе группы B.
